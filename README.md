# OptoGrid

OptoGrid is a battery-powered BLE-enabled low-cost wireless random access optogenetic stimulation device, with 9-axis (acceleration XYZ, angular veleocity XYZ, magnetic field XYZ) movement tracking capability. 

## Specification

### Weight
| Component                | Weight (g) |
|--------------------------|------------|
| Battery PCB Assembly     | 0.9        |
| Main PCB Assembly        | 0.5        |
| Housing units + magnets  | 1.0        |
| **Total weight**         | **2.4**    |

## Folder Structure

- **3D/**  
  CAD models and mechanical assemblies for OptoGrid.

- **PCB/**  
  Original PCB project file (.eprj) exported from JLCEDA Pro, a folder containing PCB design files exported to Altium Designer format, PDF of the schematics.
- **OptoGrid_Snapshots.pdf/**  
  Photos of OptoGrid V0.05 assembly.


## Purpose

This repository is to share the hardware design files of OptoGrid with OpenEphys.
